# Derecho Penal Argentino — Skills para Claude Code

Repositorio de skills, plantillas y herramientas para Claude Code 
orientadas específicamente al derecho penal argentino.

## ¿Qué es esto?

Un conjunto de herramientas que potencian Claude Code para la 
litigación penal en Argentina: análisis estratégico de causas, 
detección de vulnerabilidades, modelos de escritos y checklists 
pre-audiencia.

## Skills disponibles

| Skill | Descripción | Estado |
|-------|-------------|--------|
| `teoria-del-caso` | Análisis tripartito: defensa, fiscalía y juez | ✅ Disponible |
| `prision-preventiva` | Análisis de medidas cautelares | 🔧 En construcción |
| `recursos-casacion` | Estructura y motivos casatorios | 🔧 En construcción |
| `nulidades-procesales` | Scanner de vicios procesales | 🔧 En construcción |
| `salidas-alternativas` | Probation y juicio abreviado | 🔧 En construcción |

## Instalación

```bash
git clone https://github.com/RICARDO VERNAL/derecho-penal-argentino ~/.claude/skills/penal-argentino
```

## Jurisdicción

Las skills están calibradas para:
- **Código Penal:** Ley 11.179 y modificatorias
- **CPPN:** Ley 23.984 y modificatorias  
- **CPP Tucumán:** Ley 6203 y modificatorias
- **Jurisprudencia:** CSJN, Cámara Federal de Casación Penal
